Commands

The mod provides some commands for managing drafts.

  • /pomdrafts codes - list all draft codes.
  • /pomdrafts remap {from} {to} - remap draft code (replace existing codes for players).
  • /pomdrafts learnall [playerName] - add all available drafts for a player.
  • /pomdrafts list [playerName] - list player drafts.
  • /pomdrafts clear [playerName] - remove all player drafts.
  • /pomdrafts learn {draftCode} [playerName] - add a draft to player.
  • /pomdrafts remove {draftCode} [playerName] - remove the draft from player.
