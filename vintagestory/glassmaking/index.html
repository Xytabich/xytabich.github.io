Overview

Github page.

This modification adds the craft of glass blowing to the game: melting, blowing, molding and processing glass products.

At the moment there are few ways to create products:

  • Blowing - with the help of a special pipe and tools, you can give any rounded glass shape.
  • Molding - glass is blown or poured into a prepared mold.
  • Post-processing - using a workbench, it is possible to make more complex glass products. For example, combine several glass blanks together, or add details using other materials.

The modification also adds glass typing, so you can use many types and colors of glass in one recipe.

The product resulting from blowing or molding must be annealed to remove internal stresses.
