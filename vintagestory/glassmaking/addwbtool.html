Adding workbench tool

The mod includes a template for simple tools. This tool is placed on a workbench and can have simple particles while working.

To define a tool, add the glassmaking:workbenchtool behavior to the item.


{
	"behaviors": {
		"name": "glassmaking:workbenchtool",
		"properties": {
			"code": ...,
			"isTool": ...
		}
	},
	"attributes": {
		"workbenchParticles": [ ... ]
	}
}
  • The code parameter is the tool code used in the recipe step.
  • The isTool parameter can be set to true if you want the item's durability to be wasted. The amount of durability used can be specified in the recipe using toolDurabilityCost.
  • The workbenchParticles parameter is the particles that will be emitted during work on the workbench.
